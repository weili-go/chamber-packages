Class Block

title

Block

description

Plasma Block If we have 40bit for amount, we need 40 depth tree and 2^40 prime numbers. 2^40 is 1.0995116e+12 ETH: 1 in plasma is 1 microether Plasma's capacity is 1098756 ether ETH: 1 in plasma is 1 gwei Plasma's capacity is 1098 ether

If we have 48bit for amount, we need 48 depth tree and 2^49 prime numbers. 2^48 is 2.8147498e+14(260000000000000) ETH: 1 in plasma is 1 gwei Plasma's capacity is 260000 ether ETH: 1 in plasma is 10 gwei Plasma's capacity is 2600000 ether

When there are not enough prime numbers, operator don't receive split tx and do merge.
